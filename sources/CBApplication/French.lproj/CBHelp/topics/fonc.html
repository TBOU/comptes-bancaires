Fonctionnalités de Comptes Bancaires

L'application Comptes Bancaires propose différentes fonctionnalités destinées à faciliter la gestion de vos comptes.

Virement de compte à compte

Vous pouvez effectuer un virement entre deux comptes : un mouvement de type virement débiteur sera généré dans le compte source et un mouvement de type virement créditeur sera généré dans le compte cible.

Cliquez sur le lien ci-dessous pour en savoir plus :

Virement de compte à compte


Mouvements périodiques et écritures automatiques

L'application Comptes Bancaires permet de définir des mouvements périodiques (par exemple un prélèvement d'assurance de 50 € tous les 7 du mois). A l'ouverture d'un document, Comptes Bancaires parcourt l'ensemble des définitions de mouvements périodiques, calcule les nouvelles écritures à générer en fonction de la date du jour et les affiche dans une fenêtre : vous pouvez alors les générer immédiatement ou bien les remettre à plus tard (elle vous seront alors reproposées la prochaine fois).

Cliquez sur un lien ci-dessous pour en savoir plus :

Mouvements périodiques
Ecritures automatiques en suspens


Mise à jour multiple

La modification, le pointage/dépointage et la suppression peuvent se faire sur plusieurs mouvements en même temps.

Cliquez sur le lien ci-dessous pour en savoir plus :

Mise à jour multiple


Filtrage des mouvements

Il est possible de filtrer la liste des mouvements d'un compte : mouvements non pointés, mouvements ayant un type d'opération donné, mouvements dont le libellé ou la catégorie contient un mot-clef donné.

Cliquez sur le lien ci-dessous pour en savoir plus :

Filtrage des mouvements


Statistiques d'un compte

Les calculs de statistiques s'effectuent compte par compte. Ils permettent de visualiser (pour une période donnée) le total des débits, le total des crédits ainsi que les soldes par catégories.

Cliquez sur le lien ci-dessous pour en savoir plus :

Statistiques d'un compte


Libellés prédéfinis

Pour faciliter l'édition des mouvements, vous pouvez prédéfinir des libellés afin d'éviter de les ressaisir à chaque fois. Vous pouvez également associer un montant à un libellé prédéfini : lorsque vous sélectionnerez le libellé prédéfini lors de l'édition d'un mouvement, le montant sera valorisé automatiquement.

Cliquez sur le lien ci-dessous pour en savoir plus :

Libellés prédéfinis


Imports / Exports

Les documents peuvent être ouverts et enregistrés aux formats binaire (.cba) ou XML (.cbx). Le format XML peut être utilisé pour générer un document Comptes Bancaires à partir de sources de données externes, ou bien pour exporter les données d'un document Comptes Bancaires vers d'autres applications.

Il est aussi possible d'exporter la liste des comptes, ou bien la liste des mouvements d'un compte, dans un fichier texte. Cela peut permettre de récupérer le contenu pour le coller dans une autre application.

Cliquez sur un lien ci-dessous pour en savoir plus :

La clôture de l'exercice supprime pour l'ensemble des comptes tous les mouvements pointés antérieurs à une date de référence.
