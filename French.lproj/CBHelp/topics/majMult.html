Mise à jour multiple

La modification, le pointage/dépointage et la suppression peuvent se faire sur plusieurs mouvements en même temps.

  1. Lorsque la fenêtre de détail du compte est active, sélectionnez plusieurs mouvements (en cliquant tout en maintenant les touches Majuscule ou Commande enfoncées).
  2. Modification multiple : cliquez sur le bouton Editer (ou bien choisissez le menu Compte > Editer le(s) mouvement(s)).
  3. Pointage/dépointage multiple : appuyez sur Commande + Return (ou bien choisissez le menu Compte > Pointage / Dépointage).
  4. Suppression multiple : cliquez sur le bouton Supprimer (ou bien choisissez le menu Compte > Supprimer le(s) mouvement(s)).

La fenêtre de modification multiple possède une case à cocher en face de chaque champ. Un champ sera mis à jour uniquement si la case associée est cochée.
