Ajout d'un mouvement

Vous pouvez ajouter un nouveau mouvement à la liste des mouvements d'un compte. Les types d'opération suivants sont disponibles :

  • Carte Bleue (débit)
  • Prélèvement (débit)
  • Chèque (débit ; gestion d'un compteur automatique pour le numéro de chèque)
  • Chèque Emploi Service (débit ; gestion d'un compteur automatique pour le numéro de chèque Emploi Service)
  • Virement débiteur (débit)
  • Virement créditeur (crédit)
  • Dépôt (crédit)

  1. Lorsque la fenêtre de détail du compte est active, cliquez sur le bouton correspondant au type d'opération souhaité (ou bien choisissez le menu Compte > Ajouter un mouvement > "Type d'opération").
  2. Saisissez la date.
  3. Modifiez éventuellement l'opération a posteriori.
  4. Saisissez le libellé ou sélectionnez un libellé prédéfini.
  5. Affectez une catégorie au mouvement si vous le souhaitez.
  6. Saisissez le montant.
  7. Cliquez sur OK.
