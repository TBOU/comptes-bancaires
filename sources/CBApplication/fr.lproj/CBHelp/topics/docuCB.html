Documents Comptes Bancaires et formats associés

L'application Comptes Bancaires permet de créer de nouveaux documents ainsi que d'ouvrir, modifier et enregistrer des documents existants. La fenêtre principale d'un document est la fenêtre de liste des comptes. Pour chaque compte, une fenêtre de détail peut être ouverte.

Les documents peuvent être ouverts et enregistrés aux formats Comptes Bancaires (.cba) et Comptes Bancaires XML (.cbx). Le format binaire .cba est le format natif : c'est lui qu'il faut utiliser de préférence. Le format XML .cbx peut être utilisé pour générer un document Comptes Bancaires à partir de sources de données externes, ou bien pour exporter les données d'un document Comptes Bancaires vers d'autres applications.

  • Pour créer un nouveau document, choisissez le menu Fichier > Nouveau.
  • Pour ouvrir un document existant, choisissez le menu Fichier > Ouvrir.
  • Pour fermer le document actif, choisissez le menu Fichier > Fermer.
  • Pour sauvegarder le document actif, choisissez le menu Fichier > Enregistrer.
  • Pour sauvegarder le document actif sous un nouveau nom, choisissez le menu Fichier > Enregistrer sous (vous pouvez alors sélectionner le format de fichier).
  • Pour remettre le document actif dans l'état correspondant à la dernière version enregistrée, choisissez le menu Fichier > Version précédente.

Lors de l'ouverture d'un document XML, Comptes Bancaires recherche toute l'information disponible. Si une information est manquante, une valeur par défaut lui sera attribuée.
