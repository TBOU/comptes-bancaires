Pointage d'un mouvement

Vous pouvez pointer un mouvement afin qu'il soit pris en compte dans le calcul du solde banque.

  1. Lorsque la fenêtre de détail du compte est active, sélectionnez un mouvement.
  2. Appuyez sur Commande + Return (ou bien choisissez le menu Compte > Pointage / Dépointage).

Vous pouvez aussi cliquer directement dans la case à cocher située dans la colonne pointage.
