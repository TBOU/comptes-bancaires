Virement de compte à compte

Vous pouvez effectuer un virement entre deux comptes :

  • Un mouvement de type virement débiteur sera généré dans le compte source
  • Un mouvement de type virement créditeur sera généré dans le compte cible

  1. Lorsque la fenêtre de liste des comptes est active, cliquez sur le bouton Virement de compte à compte (ou bien choisissez le menu Liste des comptes > Virement de compte à compte).
  2. Sélectionnez le compte débité.
  3. Sélectionnez le compte crédité.
  4. Saisissez la date.
  5. Saisissez le libellé ou sélectionnez un libellé prédéfini (parmi ceux de type virement débiteur du compte source).
  6. Affectez une catégorie au mouvement si vous le souhaitez.
  7. Saisissez le montant.
  8. Cliquez sur OK.

Les comptes source et cible ne peuvent être les mêmes.
