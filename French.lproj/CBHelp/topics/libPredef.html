Libellés prédéfinis

Pour faciliter l'édition des mouvements, vous pouvez prédéfinir des libellés afin d'éviter de les ressaisir à chaque fois. Vous pouvez également associer un montant à un libellé prédéfini : lorsque vous sélectionnerez le libellé prédéfini lors de l'édition d'un mouvement, le montant sera valorisé automatiquement.

Les libellés prédéfinis sont classés par types d'opération.

  1. Lorsque la fenêtre de détail du compte est active, choisissez le menu Compte > Libellés prédéfinis > "Type d'opération".
  2. Pour ajouter un libellé prédéfini, cliquez sur le bouton "+".
  3. Pour renommer un libellé prédéfini, double-cliquez sur son titre et saisissez le nouveau libellé.
  4. Pour modifier le montant associé à un libellé prédéfini, double-cliquez sur son montant et saisissez une valeur (ou rien).
  5. Pour supprimer un libellé prédéfini, cliquez sur le bouton "-".
  6. Pour éditer les libellés prédéfinis associés à un autre type d'opération, choisissez la nouvelle opération dans le menu pop-up.
  7. Cliquez sur Fermer pour terminer.
